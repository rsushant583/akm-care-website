# AKM Care & AKM Freight

Official website for AKM Care & AKM Freight — providing
industrial training, HR solutions, logistics, and
authentic rural e-commerce across India.

## Tech Stack
- React + Vite
- Tailwind CSS
- Supabase (Database + Auth)
- Resend (Email notifications)

## Getting Started
```bash
npm install
npm run dev
```

## Environment Variables
Create a .env file:
```
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
VITE_SUPABASE_SERVICE_KEY=
VITE_RESEND_API_KEY=
VITE_YOUTUBE_API_KEY=
VITE_ADMIN_PIN=AKM2025
```

## Deployment
Deploy to Vercel. Add all env variables in Vercel dashboard.
